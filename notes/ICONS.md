# Lucide Icons Used

This document lists all Lucide React icons used in the UN80 Actions application.

## Application Icons

### Main Components

- **`ArrowUpDown`** - Sort controls
- **`Briefcase`** - Work package indicators
- **`Check`** - Selection checkmarks in filter dropdowns
- **`ChevronDown`** - Dropdown and collapsible indicators (with rotation)
- **`ChevronUp`** - Dropdown open state indicators
- **`FileText`** - Document reference indicators
- **`Info`** - Information and details buttons
- **`Layers`** - Workstream indicators
- **`ListTodo`** - Actions data card icon
- **`RotateCcw`** - Reset filters button
- **`Search`** - Search input indicators
- **`User`** - Work package lead filter icon
- **`Users`** - Multiple leads indicator

## UI Component Icons (shadcn/ui)

### From `components/ui/`

- **`CheckIcon`** - Selected items in dropdowns and selects
- **`ChevronDownIcon`** - Select triggers and accordion indicators
- **`ChevronUpIcon`** - Select scroll indicators
- **`ChevronRightIcon`** - Dropdown submenu indicators
- **`CircleIcon`** - Radio item indicators
- **`SearchIcon`** - Command palette search
- **`XIcon`** - Dialog close buttons

## Common Settings

### Sizes (via Tailwind CSS)
- `w-3 h-3` (12px) - Smallest
- `w-3.5 h-3.5` (14px) - Small
- `w-4 h-4` (16px) - Standard (most common)
- `w-5 h-5` (20px) - Medium
- `w-6 h-6` (24px) - Large

### Stroke Width
- **2px** (Lucide default)

### Colors
- `text-un-blue` - Primary brand color (#009edb)
- `text-gray-500`, `text-gray-600`, `text-gray-700` - Various gray shades
